# M&T Venezuela: Protocol Zero Brand Kit (2026)

Bienvenido a la documentación oficial del sistema de diseño **Protocol Zero**. Este standard define la identidad visual de nuestros sistemas de inteligencia, priorizando la precisión técnica sobre la decoración.

## Índice de Documentación

### [01. Filosofía: Estética Forense](./01_PHILOSOPHY.md)

- **Concepto**: Neutralidad radical y diseño invisible.
- **Principio**: La interfaz como lente óptico, no como pintura.

### [02. Sistema de Color: High-Performance](./02_COLORS.md)

- **Base**: Escala "Isotope" (Zinc) para reducción de fatiga visual.
- **Semántica**: Uso estricto de colores de estado. Sin decoración.

### [03. Tipografía & Layout: Grid 8pt](./03_TYPOGRAPHY_LAYOUT.md)

- **Estructura**: La matemática del diseño sobre la rejilla de 8 puntos.
- **Legibilidad**: Uso de Inter/SF con tracking dinámico.

### [04. Cinemática: Peso y Respuesta](./04_FLUIDITY.md)

- **Dinámica**: Física newtoniana para simular masa y resistencia.
- **Materiales**: Policarbonato estructural y cristal óptico.
